# 📝 Python To-Do List Application

A simple, interactive, and terminal-based To-Do List application built using Python. This project allows users to efficiently manage their daily tasks by adding, viewing, deleting, and tracking the status of their activities.

## 🚀 Features
- **Add Tasks:** Create new tasks with a single command.
- **View Tasks:** Display all current tasks with their status (Pending/Completed).
- **Mark Complete:** Update the status of tasks easily.
- **Delete Tasks:** Remove completed or unwanted tasks from the list.
- **Data Persistence:** Automatically saves your tasks so you don't lose data when closing the app.
